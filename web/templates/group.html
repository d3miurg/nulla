Новое оформление сайта

Да, внешне ничего не изменилось, но теперь стала доступна вкладка 'Настройки', где можно поменять оформление сайта. Для этого нужен набор CSS файлов: core.css, menu.css, news.css, chat.css и about.css. Создать их можно самому. Чтобы узнать, как должны состоять эти файлы, нужно зайти в консоль разработчика (F12). После этого нужно выбрать вкладку 'Стили', там будут показаны все файлы стилей, которые используются на этой странице. Для изучения полного списта файлов нужно проделать это на всех страницах сайта. На самом деле, достаточно одного файла из набора для нормальной работы сайта, но выглядеть с этим одним файлом он будет безобразно.
